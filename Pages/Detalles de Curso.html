Descripcion:

¿Alguna vez te preguntaste cómo se crean las páginas web que usás todos los días? Este curso es tu puerta de entrada al mundo del desarrollo web. Está pensado especialmente para personas que no tienen conocimientos previos en programación y quieren aprender desde cero cómo construir sitios web modernos, funcionales y visualmente atractivos.

A lo largo del curso, vas a descubrir cómo funciona la estructura de una página web utilizando HTML (HyperText Markup Language), el lenguaje que organiza el contenido, y cómo darle estilo y personalidad con CSS (Cascading Style Sheets), la herramienta que permite controlar colores, tipografías, márgenes, diseños y mucho más.

Este curso combina teoría clara con práctica constante, para que puedas aplicar lo aprendido desde el primer día. Vas a crear tus propios proyectos, entender cómo se conectan los distintos elementos de una página y adquirir las bases necesarias para seguir creciendo como desarrollador/a web.
